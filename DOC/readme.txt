Данная программа предназначена для учета компьютерной техники в организациях.
Программа написана на Visual Basic 6.0 SP5, соответственно для работы требуется файл msvbvm60.dll.

Поддерживается загрузка данных из AIDA32 и ASTRA

---------------------------------------------------

"База компьютеров организации @" позволяет вести базу данных инвентаризации без утомительного ручного ввода информации о компьютерах и устройствах. Это возможно благодаря поддержке импорта отчетов программ сбора информации об оборудовании: Sysinfo Lab ASTRA и AIDA32.

Sysinfo Lab ASTRA (ex-Informer) - одна из лучших на сегодняшний день программ анализа аппаратной конфигурации компьютера. Она "узнает в лицо" 132 типа процессоров и сопроцессоров, 3700 моделей мониторов, 330 производителей материнских плат, более 3100 PCI/AGP устройств, 2200 PnP устройств; измеряет тактовую частоту процессора и частоту шины с точностью 0.1МГц; определяет оригинальную частоту процессора (без учета разгона). Отчеты ASTRA содержат информацию о возможностях процессора, кэше, SPD модулях памяти, чипсете, BIOS, винчестере, CD-ROM, видеокарте, звуковой карте, сетевой карте, принтере и многом другом.
официальный сайт ASTRA  http://www.sysinfolab.com/ru

AIDA32 - профессиональный инструмент для диагностики оборудования и анализа системной конфигурации. AIDA32 работает под любыми ОС семейства Win32 и совместима как с Windows 95/98/ME, так и с Windows NT/2000/XP. Отчеты AIDA32 содержат данные об оборудовании: информация о CPU, материнской плате, памяти (в том числе количество и тип модулей), HDD (IDE и SCSI), CD-ROM, звуковой карте и других устройствах.

AIDA32 получает данные об оборудовании на низком уровне (а не только по системному реестру Windows, как большинство Win32-информеров), используя собственную базу данных (около 21 000 устройств). AIDA32 позволяет собирать информацию с удаленных компьютеров по сети TCP/IP.

AIDA32 во всех вариантах (в том числе Enterprise Edition и Network Edition) распространяется бесплатно для частных и корпоративных пользователей.
официальный сайт AIDA32 http://www.aida32.hu/

Импортировав отчет ASTRA или AIDA32 в программу, вы внесете компьютер в базу данных инвентаризации, не заполняя руками ни одной формы. Берегите время и клавиатуру - используйте программу вместе с ASTRA или AIDA32.
